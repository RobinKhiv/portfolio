Spades Spanish

While adopting a new language can be tricky, acquiring certain words in intervals can help you learn more efficiently. Spaced repetition is written using a singly link data structured algorithm. The application tests the user with a word in the desired language. Depending on if the answer is right or wrong, the word will be pushed further down the list to a certain amount of spaces.

Live | GitHub

How Does it work?

The application starts off by asking for a translation of a spanish word. If the answer is wrong the word will be tested again after the next turn. If the word is translated correctly the word will come back at a later time. By this technique, users will be able to remember words they got wrong first and words that user get right will be quizzed less frequently.

Technologies Used:

  • Front-End: HTML, CSS, Javascript, React(Router, Context), Cypress
  • Back-End: Node.js, Express, PostgreSQL, Mocha/Chai, Supertest
  • Hosting: Heroku, Zeit/Now
